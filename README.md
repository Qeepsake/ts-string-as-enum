# Convert String to Enum with TypeScript

## Install

```bash
npm install --save @qeepsake/ts-string-as-enum
```

## Usage

```ts
export declare enum Colors {
    /** JSDoc */
    Blue = "Blue",
    /** JSDoc */
    Red = "Red",
    /** JSDoc */
    Green = "Green",
}

tsStringAsEnum(Colors, 'blue'); // -> Colors.Blue
```
